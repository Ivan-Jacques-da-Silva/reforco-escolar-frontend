# Reforço Escolar — Front-end (React + Vite + Tailwind)

Projeto demo só de UI, sem backend. Componentes `@/components/ui/*` são implementações leves no estilo shadcn.

## Rodar

```bash
npm install
npm run dev
```

## Estrutura
- `src/components/ui/*` — primitivas (Button, Card, Badge, Input, Label, Table)
- `src/App.jsx` — tela principal
- `tailwind.config.js` / `postcss.config.js` — Tailwind
- `vite.config.js` — React + alias `@`
